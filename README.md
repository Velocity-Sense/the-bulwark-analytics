### Getting Started with the dbt Project

Here's how to get oriented to the project:

1. Read the entire README file (this one!)
2. Read The Bulwark Modern Data Stack Guide
3. Check out the dbt_project.yml file. There you will find where we have defined our materializations, query tags, warehouse sizes, and hooks
4. Go to [How we structure our dbt projects](https://docs.getdbt.com/best-practices/how-we-structure/1-guide-overview) for an overview of project structure
5. Go to [How we style our dbt projects](https://docs.getdbt.com/best-practices/how-we-style/0-how-we-style-our-dbt-projects) for an overview of project style
